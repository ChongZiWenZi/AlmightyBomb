# AlmightyBomb
Android消息提示框

使用步骤

**Step 1. Add the JitPack repository to your build file**
 ``` java
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

  **Step 2. Add the dependency**
  ``` java
    dependencies {
	        implementation 'com.github.ChongZiWenZi:AlmightyBomb:v1.0.0'
	  }
```
